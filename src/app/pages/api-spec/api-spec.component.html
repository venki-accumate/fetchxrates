Quickstart

Make your first API call in under 5 minutes.

1

Sign up and get your API key

Create an account at fetchxrates.com and subscribe to a plan. Your JWT token is available in the dashboard under API Settings. Tokens are issued via AWS Cognito after sign-in.

open_in_new View pricing plans
2

Make your first request

Call the /api/latest endpoint to retrieve the latest exchange rates for a base currency.

terminal cURL 
curl -X GET \
  "https://uapi.fetchxrates.com/api/latest?base=USD" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
3

Handle the response

A successful response returns 200 OK with a JSON body containing the base currency, date, and rates object.

output Response 200 OK 
{
  "base": "USD",
  "date": "2026-03-19",
  "rates": {
    "EUR": 0.9261,
    "GBP": 0.7916,
    "JPY": 149.82,
    "AUD": 1.5912,
    "CAD": 1.3571,
    "CHF": 0.8974
  }
}
4

Explore more endpoints

Now that you're up and running, explore historical rates, bulk conversion, and reference data endpoints.

Token tips

  • Tokens are time-limited — refresh them via your Cognito session before expiry.
  • Never expose your token in client-side code or public repositories.
  • Each subscription plan has a rate limit — see your dashboard for current usage.
  • A 401 Unauthorized response usually means your token has expired.
